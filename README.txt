Note: run these cmds in the folder containing all files: Library Management System

INSTALL INSTRUCTIONS: npm install (to install all node modules)

LAUNCH INSTRUCTIONS: npm start

TESTING INSTRUCTIONS: http://localhost:3000
1. Filter books by entering a title, author or genre and press search
2. Add books by pressing the "Add Book" button and filling in the appropriate fields. 
    Press the "Submit" button to finalize the book's information
    To close the menu press the "Add Book" button again
3a. You may export all library books as CSV using the "Export as CSV" button
3b. You may export all library books as JSON using the "Export as JSON" button
